Front-end

前端

前端的工作

更美观，更易用

Performance, Accuracy

Reliability

后台的工作

更准确，性能更好，服务更可靠

前端

  • 数据展现
  • 用户体验(交互,视觉)
  • 浏览器

后端

  • 数据处理，业务逻辑
  • 可用性
  • 服务器，数据库

In a word

Front-end is what you see; back-end is how it works.

后端开发技术方面

  • 编程语言

  • 数据库

  • 服务器

  • 操作系统

编程语言

PHP, Python, Ruby, Node.js, Java ...

在不同的业务场景下选择合适的语言

数据库

MySQL, SQL Server, Oracle, MongoDB, Redis ...

关系型、非关系型，都是数据的角色，以一定方式存储数据

Web服务器

Apache, Nginx, Lighthttpd, IIS ...

负责处理用户请求，提供网页

操作系统

*nix, Windows

Linux

Debian, Open Suse, Red Hat, Gentoo, Arch ...

与其说选择操作系统，不如说选择 Linux 发行版

举例(微博)

  1. 发微博
  2. 服务器接收数据
  3. 存入数据库
  4. 看微博
  5. 读取数据库取出所需内容
  6. 发送到客户端(浏览器)

编程语言是桥梁，贯穿始终，存取数据，处理业务逻辑等。
